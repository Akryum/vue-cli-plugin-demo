# vue-cli-plugin-demo

Very simple [vue cli 3](https://github.com/vuejs/vue-cli) plugin.

Install deps:

```
yarn
```

Add it in a vue cli project:

```
yarn add vue-cli-plugin-demo@file:../path-to-the-plugin
```

Invoke it:

```
vue invoke demo
```

After making changes, refresh it in the project:

```
yarn --force
```
